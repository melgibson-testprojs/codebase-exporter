# Codebase Exporter

A simple Python utility that exports a folder’s directory structure and
the contents of `.py` and `.txt` files into a single text file.

Useful for:
- Code reviews
- LLM context sharing
- Project archiving
- Documentation snapshots

---

## Features
- Recursively scans folders
- Prints directory tree structure
- Extracts contents of `.py` and `.txt` files
- Includes full file paths before each file’s content
- Outputs everything into one text file

---

## Requirements
- Python 3.7+
